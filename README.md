# CSS Flags

A collection of 193 responsive flags coded with CSS using a single HTML element.

## The Project

Coding flags for fun.


## How it works

In HTML create an element and add the class `flag` along with the name of the country (check the list for names).

```
<span class="flag name-of-country"></span>
```

Checkout the [demo page](https://alvaromontoro.github.io/CSS-Flags/demo/) to see all the flags in one place.
